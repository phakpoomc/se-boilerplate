# Project Context

This repository is a deliberately small full-stack boilerplate for student projects.

- `apps/api`: Express and TypeScript API.
- `apps/web`: React, Vite, TanStack Router/Query, Tailwind, and shadcn/ui.
- `packages/db`: Prisma, SQLite migrations, seed data, and shared client access.
- `packages/shared`: Zod schemas and contracts shared by the API and web app.

The included `showcase-items` feature is a teaching example, not a required product domain. Replace
it as the project develops while preserving clear module boundaries and beginner-friendly setup.

Treat this starter's root `AGENTS.md` as the instruction boundary, even if the folder is temporarily
copied inside another repository.
